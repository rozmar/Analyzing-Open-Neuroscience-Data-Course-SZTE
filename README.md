# Analyzing-Open-Neuroscience-Data-Course-SZTE
This course is a patch-work of already existing sources mixed with our own ideas. 
- The python basics are built on the Python bootcamp of the Allen Institute on the Dynamic Brain summer course: https://github.com/alleninstitute/python_bootcamp
- Patch-seq module is based on Ashley Juavinett's codebase (https://github.com/ajuavinett/CellTypesLesson)
- BCI module is based on the SWDB2025 at the Allen Institute (https://alleninstitute.org/events/summer-workshop-on-the-dynamic-brain-2025/)
## 2025/2026

### Tematika
- 1-6: Bevezetés az idegtudományi adathalmazokba & Python alapozás
- 7-9: Patch-seq, scRNAseq
- 10-11: Brain Computer Interface learning & all-optical circuit mapping
- 12-13: Projektek elkeszitese
