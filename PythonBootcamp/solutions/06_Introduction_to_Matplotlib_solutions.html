Introduction to Matplotlib

matplotlib is a plotting library for Python.

Pros:

  • Huge amount of functionality/options.
  • Works with numpy arrays and python lists.
  • Comes with many prepackaged Python distros (anaconda, WinPython, etc.).
  • Easily saves plots to image (.png, .bmp, etc.) and vector (.svg, .pdf, etc.) formats.
  • Has an excellent set of examples (with code) at the matplotlib gallery
  • Shares many syntactic conventions with Matlab.

Cons:

  • Slow for rapidly updating plots.
  • 3D plotting support is not great.
  • Documentation is not always useful.
  • Essentially has two interfaces. One is intended to be close to Matlab, the other is object oriented. You will find examples that assume one or the other, but rarely the one you are after.
  • Shares many syntactic conventions with Matlab.

Resources:

  • cheat sheets
  • youtube tutorials

Matplotlib Basics

We will be using pyplot.subplots (plt.subplots()) to create our beautiful data visualizations. Calling plt.subplots() gives back two objecs that are conventionally called a "figure" and an "axes". To represent these we will create two variables, fig and ax.

fig: You can think of the figure as the frame of your plot.

  • You can resize, reshape the frame but you cannot draw on it.
  • On a single notebook or a script, you can have multiple figures.
  • Each figure can have multiple subplots.

ax: an axes is your metaphorical canvas to draw on. Or rephrasing, it is the blank sheet you can plot and hold your data.

  • An axes object can only belong to one figure.

Matplotlib Code Basics

  • create a figure: fig, ax = plt.subplots()

    • Adjust figure size: fig, ax = plt.subplots(figsize = (horizontal, vertical)
    • Multiple plots per figure: fig, ax = plt.subplots(n_rows, n_columns)
      • Note!: we will cover this more in-depth later!

  • add title: ax.set_title('plot title')

  • set x label: ax.set_xlabel('x label')

  • set y label: ax.set_ylabel('y label')

NOTE: Later (tomorrow) you will learn Seaborn, which is another plotting package. Seaborn is essentially a wrapper for matplotlib there are many matplotlib conventions and functions that we will use on almost all of our plots regardless of whether they are seaborn or matplotlib plots

Image Data

Any time we have a 2D array of values, it may be useful to visualize the array as an image. Matplotlib gives us many options to control how data values are converted to image pixel colors. ​ This process is very simple with matplotlib, just use .imshow()

Basic code: ax.imshow(2d_image_array_data) ​

Helpful Parameters

  • cmap= string: The Colormap instance or registered colormap name used to map scalar data to colors. This parameter is ignored for RGB(A) data. Default: 'viridis' For more colormaps check out the colormap documentation)
  • clim =(min,max): Set the color limits of the current image. If either vmin or vmax is None, the image min/max respectively will be used for color scaling.

Multi Plot Figures

Figures with Subplots

A given figure can have more than one axis. The subplot command, which we used above, generates a single axis by default. But we can specify the number of axes that we want..

Complex Figures with Gridspec

Gridspec is useful when you have uneven subplots. It can get tricky for more complex plots, so first try to use ax.subplots() (like in the previous examples) if possible.

Subplot

.subplots: creates a figure and a set of subplots. This utility wrapper makes it convenient to create common layouts of subplots, including the enclosing figure object, in a single call.

General code example:

fig, axes = plt.subplots(nrows=2, ncols=1)
axes[0].plot()
axes[1].plot()

Optional Parameters for subplots call:

  • sharex/sharey = bool forces plots to display the same range along the x-axis or y-axis depending upon which argument you call. Default = False

gridspec: matplotlib

.subplots: creates a figure and a set of subplots. This utility wrapper makes it convenient to create common layouts of subplots, including the enclosing figure object, in a single call.

General code example:

fig = plt.figure()
gridspec_fig = fig.add_gridspec(nrows=2, ncols=2)
ax0 = fig.add_subplot(gs[0, 0]) # Top left corner
ax1 = fig.add_subplot(gs[0, 1]) # Top right corner
ax2 = fig.add_subplot(gs[1, :]) # Bottom, span entire width
    
ax0.plot()
ax1.plot()
ax2.plot()

helpful parameters:

  • sharex/sharey = bool forces plots to display the same range along the x-axis or y-axis depending upon which parameter you call. Default = False
